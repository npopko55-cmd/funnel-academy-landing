# Funnel academy — landing page

Одностраничный продающий сайт для **Funnel academy** (школа маркетинга Жени Ли-Юна).

**Live:** https://npopko55-cmd.github.io/funnel-academy/ (см. ниже про деплой)

## Что внутри

- `index.html` — единственный файл с разметкой, стилями и контентом
- Tailwind CSS подключён через CDN — никакого билда, открываешь файл и работаешь
- Шрифты: Inter + Playfair Display через Google Fonts
- FAQ-аккордеон — нативный `<details>/<summary>` (без JS)
- Видео — Kinescope embed (id `2qbax3MphDnHnBGHMcDLz8`)

## Структура страницы

1. **Hero** — заголовок «Как всегда давать *результат* клиентам», подзаголовок, 4 статистики, видео-эмбед
2. **Что изучить дальше** — 3 карточки мини-продуктов (5к ₽ каждый) + комбо «2+1» (10к ₽)
3. **Кому это будет полезно** — 4 портрета покупателя
4. **Остались вопросы** — FAQ (5 вопросов) + CTA «Написать в Telegram»
5. **Footer** — реквизиты-заглушка, ссылки на оферту/политику, контакты

## Как править

Открываешь `index.html` в любом редакторе (VS Code, Sublime, даже TextEdit). Меняешь текст, сохраняешь. Если работаешь локально — открой файл двойным кликом в браузере, чтобы увидеть результат.

## Деплой на GitHub Pages

Уже задеплоено. После любых правок:

```bash
git add index.html
git commit -m "update: <что поменял>"
git push
```

GitHub Pages автоматически пересоберёт страницу за 30-60 секунд.

## Перенос на свой домен (потом)

В настройках репозитория → Pages → Custom domain → вписать домен (например `funnel-academy.ru`). У домен-регистратора (Reg.ru / Beget) поставить CNAME-запись на `npopko55-cmd.github.io`.
