Как конвертировать в JPEG

  1. Открой нужный баннер по ссылке «Открыть» — он откроется в новой вкладке.
  2. Нажми F12 (или Cmd+Option+I на маке) — откроется DevTools.
  3. В панели Elements найди тег <div class="banner"> — это сам баннер 1200×900.
  4. Правый клик на этой строке в DevTools → Capture node screenshot.
  5. Браузер сохранит PNG ровно 1200×900 пикселей.
  6. Если нужен JPEG — кинь PNG в любой конвертер (например convertio.co) или через Preview на маке: открыл PNG → File → Export → выбрал JPEG.

Альтернатива без DevTools: установи расширение «GoFullPage» или «Awesome Screenshot» — сделает скрин страницы целиком. Минус: захватит и серый фон вокруг баннера, придётся обрезать.
